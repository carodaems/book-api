# Book API

## Onderwerp

Boeken zijn een belangrijk deel van mijn leven, daarom besloot ik ze als thema te gebruiken voor het maken van mijn API. Ik wou graag iets maken waarmee ik een persoonlijke lijst van al mijn boeken kon bijhouden. Als ik niet weet welk boek te lezen stuurt de API me een suggestie door een willekeurig boek te selecteren uit de lijst. Ik kan boeken toevoegen aan de lijst, hem opvragen, of een specifiek boek opvragen uit de lijst.
